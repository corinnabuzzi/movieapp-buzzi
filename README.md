# Movie App
Progetto: Web app multi-pagina in stile Netflix che mostri film e serie TV, partendo da HTML/CSS/JS già visti a lezione e integrando chiamate HTTP GET verso l’API pubblica di TMDB.

Tech stack: HTML, CSS, Vanilla JS
API: [TMDB](https://developer.themoviedb.org/docs/getting-started)
